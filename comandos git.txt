git clone "Agregar url" Esto sirve para bajar el repositorio a la computadora actual en la que estemos 

 Ejemplo de bajar este repositorio: git clone https://github.com/nicol16/aprendiendo-js.git  


 git status: Este comando nos indica cuales archivos han sido agregados, modificados y eliminados

 git add . : Este comando sirve para agregar todos los cambios realizados en nuestro repositorio local

 git commit -m "texto": Este comando va despues de git add y sirve para llevar los cambios realizados 
 al staging (area intermedia entre el repositorio local y el de la pagina) area

 git push: Este comando sirve para publicar los cambios al repositorio de la pagina y este es el paso final
